## 🌟 Featured Project: IMDB Sentiment Analysis (Classical ML + Transformer Explainability)

**Repository:** [IMDB_text_classification](https://github.com/ravina029/IMDB_text_classification)  
**Tech Stack:** Python · scikit-learn · Transformers · LIME · SHAP · Streamlit  
**Focus Areas:** Model Explainability · Comparative Evaluation · Deployment  

### 🔍 Overview
A complete NLP pipeline built for sentiment classification of IMDB movie reviews.  
It compares **classical ML models (Logistic Regression, Random Forest, MultinomialNB)** with a fine-tuned **DistilBERT** model.  
The project integrates **explainability techniques (LIME, SHAP)** to visualize model reasoning and decision behavior.

### 🚀 Highlights
- Trained and evaluated multiple models with TF-IDF and transformer embeddings.  
- Integrated **LIME** and **SHAP** for local interpretability visualizations.  
- Created an interactive **Streamlit web app** to demonstrate real-time predictions.  
- Added comprehensive **logging and experiment tracking**.  
- Includes a **live demo video** (see README in repo).
