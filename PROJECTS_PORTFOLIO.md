## 🌟 Featured Projects

### 1. 🩺 **Trustworthy Maternal Postpartum Health RAG**
**Repository:** [Maternal_postpartum_rag](https://github.com/ravina029/Maternal_postpartum_rag)  
**Tech Stack:** Python · Local LLMs · Sentence Transformers · LIME · SHAP · Streamlit  
**Focus Areas:** Trustworthy NLP · Explainable AI · Hallucination Detection · Safety-Aware Generation  

#### 🔍 Overview
A fully local, explainable, and safety-validated Retrieval-Augmented Generation (RAG) system designed to provide **reliable maternal and postpartum health guidance**.  
The system combines **document retrieval, grounded response generation, hallucination detection, safety scoring, and evidence traceability** to improve reliability and transparency in high-stakes healthcare question answering.

#### 🚀 Highlights
- Built a **fully local RAG pipeline** for pregnancy and postpartum guidance using local embeddings and LLM-based generation.
- Integrated **hallucination detection** through retrieval alignment, contradiction checking, and grounding signals.
- Added a **safety evaluation layer** with toxicity detection, risk pattern filtering, and fallback mechanisms for unsafe queries.
- Designed the system for **evidence-grounded, explainable responses** with transparent reasoning and per-sentence traceability.
- Structured the project as a **trustworthy NLP system** suitable for research in responsible AI, safety, and explainability.

### 2. 🎬 **Trustworthy Sentiment Classification: (Classical ML + Transformer Explainability)**
**Repository:** [IMDB_text_classification](https://github.com/ravina029/IMDB_text_classification)  
**Tech Stack:** Python · scikit-learn · Transformers · LIME · SHAP · Streamlit  
**Focus Areas:** Model Explainability · Comparative Evaluation · Deployment  

#### 🔍 Overview
A complete NLP pipeline built for sentiment classification of IMDB movie reviews.  
It compares **classical ML models (Logistic Regression, Random Forest, MultinomialNB)** with a fine-tuned **DistilBERT** model.  
The project integrates **explainability techniques (LIME, SHAP)** to visualize model reasoning and decision behavior.

#### 🚀 Highlights
- Trained and evaluated multiple models with TF-IDF and transformer embeddings.
- Integrated **LIME** and **SHAP** for local interpretability visualizations.
- Created an interactive **Streamlit web app** to demonstrate real-time predictions.
- Added comprehensive **logging and experiment tracking**.
- Includes a **live demo video** (see README in repo).
